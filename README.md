# 📊 Análise de Vendas com Python

## 📌 Sobre o Projeto

Este projeto tem como objetivo realizar o tratamento, preparação e análise exploratória de um dataset de vendas, transformando dados brutos em informações confiáveis para geração de insights de negócio.

O foco principal foi aplicar boas práticas de análise de dados, incluindo limpeza de dados, tratamento de valores ausentes, validação de informações e criação de novas variáveis para enriquecer a análise.

O projeto foi desenvolvido utilizando Python e bibliotecas de análise de dados, simulando um fluxo real de trabalho de um Analista de Dados.

---

# 🎯 Objetivos

- Realizar inspeção inicial dos dados;
- Identificar problemas de qualidade no dataset;
- Tratar valores ausentes e inconsistências;
- Padronizar informações categóricas;
- Validar a integridade dos dados;
- Criar novas variáveis para análise;
- Preparar uma base de dados limpa para geração de insights.

---

## 📊 Classificação de Vendas

Criação de uma classificação baseada no desempenho das vendas:

- Baixa;
- Média;
- Alta.

---

# 📈 Resultados Esperados

Após o tratamento e enriquecimento dos dados, o dataset está preparado para análises como:

- Produtos com maior faturamento;
- Categorias mais rentáveis;
- Evolução das vendas ao longo do tempo;
- Comportamento dos clientes;
- Análise de ticket médio;
- Criação de dashboards.
